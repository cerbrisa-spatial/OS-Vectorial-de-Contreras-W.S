# OS-Vectorial-de-Contreras-W.S
El OS vectorial simplemente ejecuta una operación geométrica. El "archivo" o la instrucción del sistema no está guardado en ningún lugar físico estático; es el resultado de la intersección de vectores en el espacio vr de coordenadas. El OS es un motor de renderizado geométrico puro, lo que reduce el consumo de procesamiento y RAM a niveles mínimos.
